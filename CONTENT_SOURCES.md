## Reconciliations

- One overview slide says “11 cutting edge companies,” while the current roster slides and logo grid name ten companies. The website uses the ten companies that are individually named and documented.
- Earlier materials included Indoor Robotics; the newer June 2026 overview replaces it in the displayed roster with Skapion. The website follows the newer roster.
- Legacy qFund copy dated Element Security to 2022. Element Security’s public company sources date its founding to 2021, so the website uses 2021.
- Financial and valuation figures in the supplied decks are intentionally excluded from the public website.
